Installationsanleitung

Für den Programmierworkshop für Frauen*

Python

Wir werden mit der Programmiersprache Python arbeiten. Die könnt ihr auf der offiziellen Webseite herunterladen und installieren. Wichtig ist, dass ihr Version 3, nicht 2 installiert. Achtung: Falls ihr bei der Installation unter Windows gefragt werdet, ob ihr Python auch dem Systempfad hinzufügen wollt, müsst ihr in der entsprechenden Box ein Häkchen setzen.

Texteditor

Ihr braucht außerdem einen Texteditor, mit dem ihr Plaintext (also keine Word-, PDF- o.ä. Dokumente) lesen und bearbeiten könnt. Wir empfehlen, dass ihr einen der folgenden Editoren bei euch installiert:

  • Notepad++ (für Windows)
  • Visual Studio Code (für alle Betriebssysteme)
  • TextMate (für Mac)
  • Sublime (für alle Betriebssysteme)
  • Atom (für alle Betriebssysteme)
  • Geany (für Windows und Linux)

Bei Fragen und/oder Problemen schreibt uns gerne eine Email. Sucht euch gerne einen zu eurem Betriebssystem passenden Editor aus. Wenn ihr eine Empfehlung braucht: Visual Studio Code.

Wir sehen uns beim Workshop :)
